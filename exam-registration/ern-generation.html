ERN Format Guide

Format Structure
[Year][Program][Type][Sequence]
• Year: Last 2 digits (e.g., 25 for 2025)
• Program: 3-letter code (e.g., ENG, CSE)
• Type: 1=UG, 2=PG
• Sequence: 3-digit number (001-999)
Examples
25ENG1001
UG Engineering Student - First in sequence
25CSE2015
PG Computer Science - 15th student
25ENG1123/A
Mid-semester joiner format
Important: Numbers are assigned in alphabetical order of student names. Same ERN is retained throughout the course (except UG → PG transition).
